#### Query


```graphql
query TxDetails($input: TransactionDetailsInput) {
  getTransactionDetails(input: $input) {
    list {
      blockID
      TxId
      sender
      toAddress
      fromAddress
      isTransferFrom
      tokenAddress
      amount
    }
  }
}
```

#### Query Variables

```graphql
{
    "input": {
        "tokenAddress": "0x7a1b5b6089a14e4f78050f0a5cc359400814f194",

}
```
