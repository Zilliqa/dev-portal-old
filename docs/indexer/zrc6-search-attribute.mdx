Find all NFTS with a given attribute.
This applies to NFTS which follow the ZRC-7 metadata standard

<TabItem value="GraphQL">

#### Query


```graphql
query {
  assets(
    input: {
      filter: {
        attributes: [
          { type: "Metaverse Id", value: "metaverse1" }
        ]
      }
    }
  ) {
    assetsList {
      contractAddress
      tokenId
      tokenUri
      minterAddress
      ownerAddress
      spenderAddress
      operatorAddress
      name
      description
      resource
      resourceMimetype
      externalUrl
      externalDescription
      attributes {
        traitType
        value
      }
      contractPaused
    }
  }
}
```
