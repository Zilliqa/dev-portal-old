#### Query


```graphql
query ContractDetails($input: TokenDetailsInput) {
  getTokenDetails(input: $input) {
    contractAddress
    tokenName
    tokenSymbol
    init_supply
    totalSupply
    standard
    decimals
    ownerAddress
    type
  }
}
```

#### Query Variables

```graphql
{
  "input": {
    "token": "0x98a71b463a33bbff650d123bf04f1f32d1c8b508"
  }
}
```
