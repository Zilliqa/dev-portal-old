<TabItem value="GraphQL">

#### Query


```graphql
query UserBalances($input: WalletBalancesInput) {
  getUserBalances(input: $input) {
    list {
      lastBlockID
      amount
      tokenAddress
      walletAddress
    }
  }
}
```

#### Query Variables

```graphql
{
  "input": {
    "wallet": "0x86f0875da311ee5332d5d4a8485c8d3db0f7f57b"
  }
}
```

#### HTTP Headers

```graphql

  "Authorization": "Bearer <insert token>" 
}
```
