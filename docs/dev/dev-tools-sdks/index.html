Other SDKs

Zilliqa has a number of SDKs implemented in various programming languages. If yours is not listed here, please create a PR on this repository.

zilliqa-js, gozilliqa (formerly laksago) and laksaj are active SDKs maintained by the Zilliqa core team but the zilliqa community members also actively maintain multiple SDKs.
