GetTransaction

Returns the details of a specified Transaction.

Example Request#

  • cURL
  • node.js
  • java
  • python
  • go
curl -d '{
"id": "1",
"jsonrpc": "2.0",
"method": "GetTransaction",
"params": ["cd8727674bc05e0ede405597a218164e1c13c7103b9d0ba43586785f3d8cede5"]
}' -H "Content-Type: application/json" -X POST "https://api.zilliqa.com/"
