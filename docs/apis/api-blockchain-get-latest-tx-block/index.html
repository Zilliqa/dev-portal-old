GetLatestTxBlock

Returns the details of the most recent Transaction block.

Example Request#

  • cURL
  • node.js
  • java
  • python
  • go
curl -d '{
"id": "1",
"jsonrpc": "2.0",
"method": "GetLatestTxBlock",
"params": [""]
}' -H "Content-Type: application/json" -X POST "https://api.zilliqa.com/"
