GetStateProof

Returns the state proof for the corresponding TxBlock for a smart contract. Please note that this API only serve state proof for the last 50 TxBlock.

This API is available from Zilliqa V8.1.0 onwards.

Example Request#

  • cURL
  • go
curl -d '{
"id": "1",
"jsonrpc": "2.0",
"method": "GetStateProof",
"params": ["6d84363526a2d764835f8cf52dfeefe80a360fac","A0BD91DE66D97E6930118179BA4F1836C366C4CB3309A6B354D26F52ABB2AAC6","39"]
}' -H "Content-Type: application/json" -X POST "https://api.zilliqa.com/"

HTTP Request#

Chain(s)URL(s)
Zilliqa mainnethttps://api.zilliqa.com/
Developer testnethttps://dev-api.zilliqa.com/
Local testnethttp://localhost:4201/
Isolated serverhttps://zilliqa-isolated-server.zilliqa.com/

Arguments#

ParameterTypeRequiredDescription
idstringRequired"1"
jsonrpcstringRequired"2.0"
methodstringRequired"GetStateProof"
paramsarrayRequired"Proof Parameters"

Proof Parameters#

ParameterTypeRequiredDescription
AddressstringRequiredA smart contract address of 20 bytes.
Variable HashstringRequiredSHA256 hash of the state variable
TxBlockstringRequiredCan be a valid number or latest if looking for latest TxBlock
